# Generalized Data Structure Library

A **Generalized Data Structure Library** that provides reusable implementations of fundamental data structures.  
This project demonstrates a strong understanding of **data structures, algorithm design, pointer manipulation, and modular programming**.

The goal of this repository is to build **clean, reusable, and efficient implementations** of commonly used data structures that can be easily integrated into different applications and projects.

---

## 🚀 Features

- Clean and modular implementation
- Easy-to-understand code
- Covers important core data structures
- Designed for learning and practical use
- Reusable library-style implementation

---

## 📚 Implemented Data Structures

### 1. Singly Linear Linked List

A linked list where each node contains data and a pointer to the next node in a linear sequence.

**Operations Included**

- Insert First
- Insert Last
- Insert At Position
- Delete First
- Delete Last
- Delete At Position
- Display
- Count Nodes

---

### 2. Singly Circular Linked List

A variation of a singly linked list where the last node points back to the first node, forming a circular structure.

**Operations Included**

- Insert First
- Insert Last
- Insert At Position
- Delete First
- Delete Last
- Delete At Position
- Display
- Count Nodes

---

### 3. Doubly Linear Linked List

A linked list where each node contains:

- A pointer to the previous node
- A pointer to the next node

This allows traversal in both forward and backward directions.

**Operations Included**

- Insert First
- Insert Last
- Insert At Position
- Delete First
- Delete Last
- Delete At Position
- Display
- Count Nodes

---

### 4. Doubly Circular Linked List

A circular variation of a doubly linked list where:

- The last node points to the first node
- The first node points to the last node

**Operations Included**

- Insert First
- Insert Last
- Insert At Position
- Delete First
- Delete Last
- Delete At Position
- Display
- Count Nodes

---

### 5. Stack

Stack is a **Last In First Out (LIFO)** data structure.

**Operations Included**

- Push
- Pop
- Peek
- Display
- Count

**Applications**

- Function calls
- Expression evaluation
- Undo operations
- Backtracking algorithms

---

### 6. Queue

Queue is a **First In First Out (FIFO)** data structure.

**Operations Included**

- Enqueue
- Dequeue
- Display
- Count

**Applications**

- Scheduling systems
- Process management
- Data buffering
- Task queues
